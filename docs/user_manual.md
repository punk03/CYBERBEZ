# Пользовательское руководство PROKVANT

## Введение

Добро пожаловать в систему PROKVANT - комплексную систему анализа логов и защиты энергосистем.

## Начало работы

### Доступ к системе

1. Откройте веб-браузер
2. Перейдите по адресу: `http://your-server:3000`
3. Войдите в систему (если требуется аутентификация)

## Основные функции

### Dashboard (Дашборд)

Главная страница системы показывает:
- Общее количество обнаруженных угроз
- Статистику по типам атак
- Распределение по критичности
- Последние обнаруженные угрозы

### Threats (Угрозы)

Страница управления угрозами позволяет:
- Просматривать все обнаруженные угрозы
- Фильтровать по типу атаки и критичности
- Просматривать детали каждой угрозы
- Видеть статус автоматизации

### Logs (Логи)

Страница просмотра логов:
- Просмотр всех собранных логов
- Фильтрация по источнику
- Поиск по содержимому
- Экспорт логов

### Automation (Автоматизация)

Управление автоматизацией:
- Просмотр статуса автоматизации
- Управление ожидающими подтверждения действиями
- Одобрение/отклонение действий
- Просмотр истории автоматизации

### Settings (Настройки)

Настройки системы (в разработке)

## Работа с угрозами

### Просмотр угроз

1. Перейдите на страницу "Threats"
2. Используйте фильтры для поиска нужных угроз
3. Нажмите на угрозу для просмотра деталей

### Реагирование на угрозы

Система автоматически применяет меры изоляции при обнаружении угроз. Для критических действий требуется подтверждение оператора.

## Работа с логами

### Просмотр логов

1. Перейдите на страницу "Logs"
2. Используйте фильтры для поиска
3. Просматривайте логи в реальном времени

### Отправка логов через API

```bash
curl -X POST http://your-server:8000/api/v1/logs \
  -H "Content-Type: application/json" \
  -d '{
    "log": "{\"message\": \"test\", \"level\": \"INFO\"}",
    "source": "api",
    "format": "json"
  }'
```

## Управление автоматизацией

### Подтверждение действий

1. Перейдите на страницу "Automation"
2. Просмотрите ожидающие подтверждения действия
3. Нажмите "Approve" или "Reject"
4. При необходимости укажите комментарий

## Язык интерфейса

Система поддерживает два языка:
- Русский
- Английский

Переключение языка доступно в боковом меню.

## Поддержка

При возникновении проблем обращайтесь к администратору системы.
